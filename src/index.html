Interactive Circuit Solver
Resistance:
Voltage:

  1. Set values, then drag and drop your components onto the grid.
  2. Hit "Wire!" to toggle wire drawing, and "Delete" to drag-delete mistakes.
  3. Results are displayed by hovering over components!

  • Only 3 way nodes are permitted
  • Only one voltage source is allowed on the grid
  • Any component can only have 2 connections!

Known bugs and improvements:

  • UI is ugly
  • Dragging functions are pretty low fidelity, leading to more bugs
  • Dragging components clash with other parts of the page, leading to more bugs
  • Multiple submits and some cases like short circuits are not supported
